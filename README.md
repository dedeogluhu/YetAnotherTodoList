## Project setup
```
npm install
```

or 

```
yarn install
```

### Compiles and hot-reloads for development
```
yarn dev
```

### Compiles and minifies for production
```
yarn build
```

### Starts deployed app
```
yarn start
```

### Important

You need to set up MONGO_URI and PORT as environment variables to work with this
